## Level 0 → Level 1

## Objective
Log into the Bandit server via SSH and find the password for level 1. 
## Commands Used
```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
ls
cat readme
```

## Why It Worked
The linux concepts that made this work are ssh, ls, and cat. Ssh is used to connect to a server, ls to list files in the current directory within the server, and cat to read/print the contents.

##what I Learned
I learned to sign onto the Bandit server in order to start my OverTheWire learning journey. I also learned the commands ssh, ls, and cat. Ssh allows me to securely and remotely connect to servers over a network. Basic linux file navigation uses ls to locate files and cat read their contents.
